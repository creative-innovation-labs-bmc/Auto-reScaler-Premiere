How to Install Auto reScaler (v1.0.0)

What is this file?
This .ZXP file is an official Adobe Extension installer. To install it safely into Premiere Pro, you just need a free ZXP manager.

---------------------------------------------

Step 1: Get a ZXP Installer
If you don't already have one, download one of the industry-standard free installers:

Anastasiy’s Extension Manager (Highly Recommended)
https://install.anastasiy.com/

aescripts + aeplugins ZXP Installer
https://aescripts.com/learn/zxp-installer/

---------------------------------------------

Step 2: Install the Extension

Close Adobe Premiere Pro.
Open your chosen ZXP Installer.
Simply drag and drop the Auto_reScaler_v1.0.0.zxp file directly into the installer window (or click "Install" and select the file).
Wait for the "Installation Successful" message.

---------------------------------------------

Step 3: Open in Premiere Pro

Launch Adobe Premiere Pro.
Open any project.
In the top menu bar, go to Window > Extensions > Auto reScaler.
The panel will open! You can dock it anywhere in your workspace.
